# container_diy
一些特殊的容器

## 性能
数据存储在连续的内存块中，读取运算效率高，与C内置的数组相当，区别是
* 可以连续追加无穷多个元素，但是只能读取最新的有限个数的元素(length)；
* 追加元素的时间开销低，不创建新的堆，只见目前的栈中寻址处理。

## 简介

* TList<Type, length>
 
  存储时间序列数据。特点的在队列之首追加元素，下脚标计数从[0]开始，
  [0]代表最新的一个元素

* List<Type, length>
 
  存储时间序列数据。特点的在队列之尾追加元素，下脚标计数从[0]开始，
  [0]代表保存的最旧的数据
